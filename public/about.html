About DNose

DNose is an automated test smell detection system designed specifically for the Dart language. It helps developers maintain code quality by identifying patterns in test code that may indicate problems.

Detectable Smells

  • Assertion Roulette
  • Conditional Test Logic
  • Duplicate Assert
  • Empty Test
  • Exception Handling
  • Ignored Test
  • Magic Number
  • Mystery Guest
  • Print Statement
  • Resource Optimism
  • Sleepy Test
  • Sensitive Equality
  • Test Without Description
  • Unknown Test
  • Verbose Test
